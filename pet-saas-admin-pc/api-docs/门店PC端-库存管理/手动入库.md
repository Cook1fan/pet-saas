

## 手动入库


**接口地址**:`/api/pc/stock/in`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:


**请求示例**:


```javascript
{
  "skuId": 0,
  "num": 0,
  "remark": ""
}
```


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|stockManualInReq|手动入库请求|body|true|StockManualInReq|StockManualInReq|
|&emsp;&emsp;skuId|SKU ID||true|integer(int64)||
|&emsp;&emsp;num|数量||true|integer(int32)||
|&emsp;&emsp;remark|备注||false|string||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|RStockRecordVO|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||StockRecordVO|StockRecordVO|
|&emsp;&emsp;id|主键 ID|integer(int64)||
|&emsp;&emsp;skuId|商品SKU ID|integer(int64)||
|&emsp;&emsp;type|变动类型|integer(int32)||
|&emsp;&emsp;changeNum|变动数量（正数入库，负数出库）|integer(int32)||
|&emsp;&emsp;beforeStock|变动前库存|integer(int32)||
|&emsp;&emsp;afterStock|变动后库存|integer(int32)||
|&emsp;&emsp;batchNo|批次号|string||
|&emsp;&emsp;relatedType|关联单据类型|string||
|&emsp;&emsp;relatedId|关联单据 ID|integer(int64)||
|&emsp;&emsp;relatedNo|关联单据号|string||
|&emsp;&emsp;remark|备注|string||
|&emsp;&emsp;createTime|创建时间|string(date-time)||
|&emsp;&emsp;createUser|操作人 ID|integer(int64)||
|success||boolean||


**响应示例**:
```javascript
{
	"code": 0,
	"message": "",
	"data": {
		"id": 0,
		"skuId": 0,
		"type": 0,
		"changeNum": 0,
		"beforeStock": 0,
		"afterStock": 0,
		"batchNo": "",
		"relatedType": "",
		"relatedId": 0,
		"relatedNo": "",
		"remark": "",
		"createTime": "",
		"createUser": 0
	},
	"success": true
}
```
