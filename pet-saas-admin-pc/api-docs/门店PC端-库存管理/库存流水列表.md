

## 库存流水列表


**接口地址**:`/api/pc/stock/records`


**请求方式**:`GET`


**请求数据类型**:`application/x-www-form-urlencoded`


**响应数据类型**:`*/*`


**接口描述**:


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|skuId|SKU ID|query|false|integer(int64)||
|type|变动类型|query|false|integer(int32)||
|batchNo|批次号|query|false|string||
|startTime|开始时间|query|false|string(date-time)||
|endTime|结束时间|query|false|string(date-time)||
|pageNum|页码|query|false|integer(int32)||
|pageSize|每页条数|query|false|integer(int32)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|RPageResultStockRecordVO|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||PageResultStockRecordVO|PageResultStockRecordVO|
|&emsp;&emsp;records|数据列表|array|StockRecordVO|
|&emsp;&emsp;&emsp;&emsp;id|主键 ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;skuId|商品SKU ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;type|变动类型|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;changeNum|变动数量（正数入库，负数出库）|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;beforeStock|变动前库存|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;afterStock|变动后库存|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;batchNo|批次号|string||
|&emsp;&emsp;&emsp;&emsp;relatedType|关联单据类型|string||
|&emsp;&emsp;&emsp;&emsp;relatedId|关联单据 ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;relatedNo|关联单据号|string||
|&emsp;&emsp;&emsp;&emsp;remark|备注|string||
|&emsp;&emsp;&emsp;&emsp;createTime|创建时间|string(date-time)||
|&emsp;&emsp;&emsp;&emsp;createUser|操作人 ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;barcode|商品条码|string||
|&emsp;&emsp;&emsp;&emsp;specName|规格名称|string||
|&emsp;&emsp;&emsp;&emsp;specValue|规格值|string||
|&emsp;&emsp;&emsp;&emsp;goodsName|商品名称|string||
|&emsp;&emsp;&emsp;&emsp;mainImage|商品主图URL|string||
|&emsp;&emsp;total|总条数|integer(int64)||
|&emsp;&emsp;current|当前页|integer(int64)||
|&emsp;&emsp;size|每页条数|integer(int64)||
|&emsp;&emsp;pages|总页数|integer(int64)||
|success||boolean||
