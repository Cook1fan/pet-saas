

## 库存预警列表


**接口地址**:`/api/pc/goods/warnList`


**请求方式**:`GET`


**请求数据类型**:`application/x-www-form-urlencoded`


**响应数据类型**:`*/*`


**接口描述**:


**请求参数**:


**请求参数**:


暂无


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|RListGoodsSkuVO|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||array|GoodsSkuVO|
|&emsp;&emsp;id|SKU ID|integer(int64)||
|&emsp;&emsp;goodsId|商品ID|integer(int64)||
|&emsp;&emsp;specName|规格名称|string||
|&emsp;&emsp;specValue|规格值|string||
|&emsp;&emsp;price|售价|number||
|&emsp;&emsp;costPrice|成本价（预留）|number||
|&emsp;&emsp;stock|库存：-1表示不限库存（服务类商品用）|integer(int32)||
|&emsp;&emsp;warnStock|库存预警值（实物商品用）|integer(int32)||
|&emsp;&emsp;barcode|商品条码（预留）|string||
|&emsp;&emsp;status|状态：0-禁用，1-启用|integer(int32)||
|&emsp;&emsp;createTime|创建时间|string(date-time)||
|success||boolean||
