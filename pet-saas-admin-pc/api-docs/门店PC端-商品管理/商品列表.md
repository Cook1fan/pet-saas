

## 商品列表


**接口地址**:`/api/pc/goods/list`


**请求方式**:`GET`


**请求数据类型**:`application/x-www-form-urlencoded`


**响应数据类型**:`*/*`


**接口描述**:


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|categoryId|分类ID|query|false|integer(int64)||
|keyword|商品名称（模糊搜索）|query|false|string||
|status|状态：0-下架，1-上架|query|false|integer(int32)||
|pageNum|页码|query|false|integer(int32)||
|pageSize|每页条数|query|false|integer(int32)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|RPageResultGoodsVO|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||PageResultGoodsVO|PageResultGoodsVO|
|&emsp;&emsp;records|数据列表|array|GoodsVO|
|&emsp;&emsp;&emsp;&emsp;id|商品ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;tenantId|租户ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;categoryId|分类ID，关联goods_category|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;categoryName|分类名称|string||
|&emsp;&emsp;&emsp;&emsp;goodsName|商品名称|string||
|&emsp;&emsp;&emsp;&emsp;mainImage|商品主图URL|string||
|&emsp;&emsp;&emsp;&emsp;isService|是否服务类商品：0-实物商品，1-服务商品|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;status|状态：0-下架，1-上架|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;version|乐观锁版本号|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;skuList|SKU列表|array|GoodsSkuVO|
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;id|SKU ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;goodsId|商品ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;skuCode|系统内部SKU编码|string||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;specName|规格名称|string||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;specValue|规格值|string||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;price|售价|number||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;costPrice|成本价|number||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;stock|实际库存|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;isUnlimitedStock|是否无限库存：0-否，1-是|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;reservedStock|占用库存（在途库存）|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;availableStock|可用库存（计算字段：stock - reservedStock）|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;barcode|商品条码|string||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;status|状态：0-禁用，1-启用|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;createTime|创建时间|string(date-time)||
|&emsp;&emsp;&emsp;&emsp;createTime|创建时间|string(date-time)||
|&emsp;&emsp;total|总条数|integer(int64)||
|&emsp;&emsp;current|当前页|integer(int64)||
|&emsp;&emsp;size|每页条数|integer(int64)||
|&emsp;&emsp;pages|总页数|integer(int64)||
|success||boolean||
