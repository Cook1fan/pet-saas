

## 商品详情


**接口地址**:`/api/pc/goods/{goodsId}`


**请求方式**:`GET`


**请求数据类型**:`application/x-www-form-urlencoded`


**响应数据类型**:`*/*`


**接口描述**:


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|goodsId||path|true|integer(int64)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|RGoodsVO|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||GoodsVO|GoodsVO|
|&emsp;&emsp;id|商品ID|integer(int64)||
|&emsp;&emsp;tenantId|租户ID|integer(int64)||
|&emsp;&emsp;categoryId|分类ID，关联goods_category|integer(int64)||
|&emsp;&emsp;categoryName|分类名称|string||
|&emsp;&emsp;goodsName|商品名称|string||
|&emsp;&emsp;mainImage|商品主图URL|string||
|&emsp;&emsp;isService|是否服务类商品：0-实物商品，1-服务商品|integer(int32)||
|&emsp;&emsp;status|状态：0-下架，1-上架|integer(int32)||
|&emsp;&emsp;skuList|SKU列表|array|GoodsSkuVO|
|&emsp;&emsp;&emsp;&emsp;id|SKU ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;goodsId|商品ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;specName|规格名称|string||
|&emsp;&emsp;&emsp;&emsp;specValue|规格值|string||
|&emsp;&emsp;&emsp;&emsp;price|售价|number||
|&emsp;&emsp;&emsp;&emsp;costPrice|成本价（预留）|number||
|&emsp;&emsp;&emsp;&emsp;stock|库存：-1表示不限库存（服务类商品用）|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;warnStock|库存预警值（实物商品用）|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;barcode|商品条码（预留）|string||
|&emsp;&emsp;&emsp;&emsp;status|状态：0-禁用，1-启用|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;createTime|创建时间|string(date-time)||
|&emsp;&emsp;createTime|创建时间|string(date-time)||
|success||boolean||
