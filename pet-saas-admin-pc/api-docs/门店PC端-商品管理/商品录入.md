

## 商品录入


**接口地址**:`/api/pc/goods/save`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:


**请求示例**:


```javascript
{
  "id": 0,
  "categoryId": 0,
  "goodsName": "",
  "mainImage": "",
  "isService": 0,
  "status": 0,
  "version": 0,
  "skuList": [
    {
      "id": 0,
      "specName": "",
      "specValue": "",
      "price": 0,
      "costPrice": 0,
      "isUnlimitedStock": 0,
      "barcode": "",
      "status": 0
    }
  ]
}
```


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|goodsSaveReq|商品保存请求|body|true|GoodsSaveReq|GoodsSaveReq|
|&emsp;&emsp;id|商品ID（更新时必填）||false|integer(int64)||
|&emsp;&emsp;categoryId|分类ID，关联goods_category||true|integer(int64)||
|&emsp;&emsp;goodsName|商品名称||true|string||
|&emsp;&emsp;mainImage|商品主图URL||false|string||
|&emsp;&emsp;isService|是否服务类商品：0-实物商品，1-服务商品||true|integer(int32)||
|&emsp;&emsp;status|状态：0-下架，1-上架||false|integer(int32)||
|&emsp;&emsp;version|乐观锁版本号（编辑时必填）||false|integer(int32)||
|&emsp;&emsp;skuList|SKU列表||true|array|GoodsSkuSaveReq|
|&emsp;&emsp;&emsp;&emsp;id|SKU ID（更新时必填）||false|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;specName|规格名称||true|string||
|&emsp;&emsp;&emsp;&emsp;specValue|规格值||true|string||
|&emsp;&emsp;&emsp;&emsp;price|售价||true|number||
|&emsp;&emsp;&emsp;&emsp;costPrice|成本价||true|number||
|&emsp;&emsp;&emsp;&emsp;isUnlimitedStock|是否无限库存：0-否，1-是||false|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;barcode|商品条码||false|string||
|&emsp;&emsp;&emsp;&emsp;status|状态：0-禁用，1-启用||false|integer(int32)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|RVoid|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||object||
|success||boolean||


**响应示例**:
```javascript
{
	"code": 0,
	"message": "",
	"data": {},
	"success": true
}
```
