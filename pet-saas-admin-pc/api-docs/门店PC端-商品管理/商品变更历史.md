

## 商品变更历史


**接口地址**:`/api/pc/goods/change-log`


**请求方式**:`GET`


**请求数据类型**:`application/x-www-form-urlencoded`


**响应数据类型**:`*/*`


**接口描述**:


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|dataType|数据类型：1-goods商品，2-goods_sku商品规格|query|false|integer(int32)||
|dataId|数据ID|query|false|integer(int64)||
|batchNo|批次号|query|false|string||
|pageNum|页码|query|false|integer(int32)||
|pageSize|每页条数|query|false|integer(int32)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|RPageResultGoodsChangeLogVO|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||PageResultGoodsChangeLogVO|PageResultGoodsChangeLogVO|
|&emsp;&emsp;records|数据列表|array|GoodsChangeLogVO|
|&emsp;&emsp;&emsp;&emsp;id|主键 ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;dataType|数据类型：1-goods商品，2-goods_sku商品规格|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;dataId|数据 ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;changeType|变更类型：1-新增，2-修改，3-删除|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;fieldName|变更字段名（已废弃）|string||
|&emsp;&emsp;&emsp;&emsp;beforeValue|变更前值（JSON格式，仅包含变更字段）|string||
|&emsp;&emsp;&emsp;&emsp;afterValue|变更后值（JSON格式，仅包含变更字段）|string||
|&emsp;&emsp;&emsp;&emsp;batchNo|批次号|string||
|&emsp;&emsp;&emsp;&emsp;remark|备注|string||
|&emsp;&emsp;&emsp;&emsp;createTime|创建时间|string(date-time)||
|&emsp;&emsp;&emsp;&emsp;createUser|操作人 ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;changeDetails|格式化后的变更详情列表|array|GoodsChangeDetailVO|
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;fieldName|字段名（英文）|string||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;fieldLabel|字段标签（中文）|string||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;beforeValue|变更前值（格式化后）|string||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;afterValue|变更后值（格式化后）|string||
|&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;skuIdentifier|SKU标识（仅SKU变更时有值，格式：规格名-规格值）|string||
|&emsp;&emsp;total|总条数|integer(int64)||
|&emsp;&emsp;current|当前页|integer(int64)||
|&emsp;&emsp;size|每页条数|integer(int64)||
|&emsp;&emsp;pages|总页数|integer(int64)||
|success||boolean||
