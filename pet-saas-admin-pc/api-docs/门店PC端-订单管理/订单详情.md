

## 订单详情


**接口地址**:`/api/pc/order/detail/{orderId}`


**请求方式**:`GET`


**请求数据类型**:`application/x-www-form-urlencoded`


**响应数据类型**:`*/*`


**接口描述**:


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|orderId|订单ID|path|true|integer(int64)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|ROrderDetailResp|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||OrderDetailResp|OrderDetailResp|
|&emsp;&emsp;order||OrderInfoVO|OrderInfoVO|
|&emsp;&emsp;&emsp;&emsp;id|订单ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;orderNo|订单号|string||
|&emsp;&emsp;&emsp;&emsp;memberId|会员ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;totalAmount|订单总金额|number||
|&emsp;&emsp;&emsp;&emsp;payAmount|实付金额|number||
|&emsp;&emsp;&emsp;&emsp;payType|支付方式：1-微信，2-现金，3-储值，4-次卡|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;payStatus|支付状态：0-待支付，1-已支付，2-已退款|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;payTime|支付时间|string(date-time)||
|&emsp;&emsp;&emsp;&emsp;transactionId|第三方交易流水号|string||
|&emsp;&emsp;&emsp;&emsp;createTime|创建时间|string(date-time)||
|&emsp;&emsp;items|订单项列表|array|OrderItemVO|
|&emsp;&emsp;&emsp;&emsp;id|订单项ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;orderId|订单ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;goodsId|商品/服务ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;goodsName|商品/服务名称|string||
|&emsp;&emsp;&emsp;&emsp;num|数量|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;price|单价|number||
|success||boolean||
