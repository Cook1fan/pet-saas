

## 开单收银


**接口地址**:`/api/pc/order/create`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:


**请求示例**:


```javascript
{
  "memberId": 0,
  "items": [
    {
      "skuId": 0,
      "num": 0
    }
  ],
  "payType": 0,
  "cardId": 0
}
```


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|createOrderReq|开单收银请求|body|true|CreateOrderReq|CreateOrderReq|
|&emsp;&emsp;memberId|会员ID||false|integer(int64)||
|&emsp;&emsp;items|商品/服务列表||true|array|OrderItemRequest|
|&emsp;&emsp;&emsp;&emsp;skuId|商品SKU ID||true|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;num|数量||true|integer(int32)||
|&emsp;&emsp;payType|支付方式：1-微信，2-现金，3-储值，4-次卡||true|integer(int32)||
|&emsp;&emsp;cardId|次卡ID（次卡支付时必填）||false|integer(int64)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|ROrderInfoVO|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||OrderInfoVO|OrderInfoVO|
|&emsp;&emsp;id|订单ID|integer(int64)||
|&emsp;&emsp;orderNo|订单号|string||
|&emsp;&emsp;memberId|会员ID|integer(int64)||
|&emsp;&emsp;totalAmount|订单总金额|number||
|&emsp;&emsp;payAmount|实付金额|number||
|&emsp;&emsp;payType|支付方式：1-微信，2-现金，3-储值，4-次卡|integer(int32)||
|&emsp;&emsp;payStatus|支付状态：0-待支付，1-已支付，2-已退款|integer(int32)||
|&emsp;&emsp;payTime|支付时间|string(date-time)||
|&emsp;&emsp;transactionId|第三方交易流水号|string||
|&emsp;&emsp;createTime|创建时间|string(date-time)||
|success||boolean||


**响应示例**:
```javascript
{
	"code": 0,
	"message": "",
	"data": {
		"id": 0,
		"orderNo": "",
		"memberId": 0,
		"totalAmount": 0,
		"payAmount": 0,
		"payType": 0,
		"payStatus": 0,
		"payTime": "",
		"transactionId": "",
		"createTime": ""
	},
	"success": true
}
```
