

## 会员列表


**接口地址**:`/api/pc/member/list`


**请求方式**:`GET`


**请求数据类型**:`application/x-www-form-urlencoded`


**响应数据类型**:`*/*`


**接口描述**:


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|keyword|手机号/姓名（模糊搜索）|query|false|string||
|tag|标签|query|false|string||
|pageNum|页码|query|false|integer(int32)||
|pageSize|每页条数|query|false|integer(int32)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|RPageResultMemberVO|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||PageResultMemberVO|PageResultMemberVO|
|&emsp;&emsp;records|数据列表|array|MemberVO|
|&emsp;&emsp;&emsp;&emsp;id|会员ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;phone|手机号|string||
|&emsp;&emsp;&emsp;&emsp;name|姓名|string||
|&emsp;&emsp;&emsp;&emsp;tags|标签（多个用逗号分隔）|string||
|&emsp;&emsp;&emsp;&emsp;createTime|创建时间|string(date-time)||
|&emsp;&emsp;total|总条数|integer(int64)||
|&emsp;&emsp;current|当前页|integer(int64)||
|&emsp;&emsp;size|每页条数|integer(int64)||
|&emsp;&emsp;pages|总页数|integer(int64)||
|success||boolean||


**响应示例**:
```javascript
{
	"code": 0,
	"message": "",
	"data": {
		"records": [
			{
				"id": 0,
				"phone": "",
				"name": "",
				"tags": "",
				"createTime": ""
			}
		],
		"total": 0,
		"current": 0,
		"size": 0,
		"pages": 0
	},
	"success": true
}
```
