

## 会员编辑


**接口地址**:`/api/pc/member/update`


**请求方式**:`PUT`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:


**请求示例**:


```javascript
{
  "id": 0,
  "phone": "",
  "name": "",
  "tags": "",
  "pets": [
    {
      "id": 0,
      "createTime": "",
      "createUser": 0,
      "updateTime": "",
      "updateUser": 0,
      "isDeleted": 0,
      "tenantId": 0,
      "memberId": 0,
      "name": "",
      "breed": "",
      "birthday": "",
      "gender": 0,
      "vaccineTime": "",
      "dewormTime": "",
      "washTime": "",
      "nextRemindTime": ""
    }
  ]
}
```


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|updateMemberReq|会员编辑请求|body|true|UpdateMemberReq|UpdateMemberReq|
|&emsp;&emsp;id|会员ID||true|integer(int64)||
|&emsp;&emsp;phone|手机号||false|string||
|&emsp;&emsp;name|姓名||false|string||
|&emsp;&emsp;tags|标签（多个用逗号分隔）||false|string||
|&emsp;&emsp;pets|宠物列表（不传则不修改，传空列表则删除所有宠物）||false|array|PetInfo|
|&emsp;&emsp;&emsp;&emsp;id|||false|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;createTime|||false|string(date-time)||
|&emsp;&emsp;&emsp;&emsp;createUser|||false|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;updateTime|||false|string(date-time)||
|&emsp;&emsp;&emsp;&emsp;updateUser|||false|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;isDeleted|||false|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;tenantId|||false|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;memberId|||false|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;name|||false|string||
|&emsp;&emsp;&emsp;&emsp;breed|||false|string||
|&emsp;&emsp;&emsp;&emsp;birthday|||false|string(date)||
|&emsp;&emsp;&emsp;&emsp;gender|||false|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;vaccineTime|||false|string(date)||
|&emsp;&emsp;&emsp;&emsp;dewormTime|||false|string(date)||
|&emsp;&emsp;&emsp;&emsp;washTime|||false|string(date)||
|&emsp;&emsp;&emsp;&emsp;nextRemindTime|||false|string(date-time)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|RMemberVO|

| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||MemberVO|MemberVO|
|&emsp;&emsp;id|会员ID|integer(int64)||
|&emsp;&emsp;phone|手机号|string||
|&emsp;&emsp;name|姓名|string||
|&emsp;&emsp;tags|标签（多个用逗号分隔）|string||
|&emsp;&emsp;createTime|创建时间|string(date-time)||
|&emsp;&emsp;pets|宠物列表|array|PetInfoVO|
|&emsp;&emsp;&emsp;&emsp;id|宠物ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;memberId|会员ID|integer(int64)||
|&emsp;&emsp;&emsp;&emsp;name|宠物名|string||
|&emsp;&emsp;&emsp;&emsp;breed|品种|string||
|&emsp;&emsp;&emsp;&emsp;birthday|生日|string(date)||
|&emsp;&emsp;&emsp;&emsp;gender|性别：1-公，2-母|integer(int32)||
|&emsp;&emsp;&emsp;&emsp;vaccineTime|疫苗日期|string(date)||
|&emsp;&emsp;&emsp;&emsp;dewormTime|驱虫日期|string(date)||
|&emsp;&emsp;&emsp;&emsp;washTime|洗护日期|string(date)||
