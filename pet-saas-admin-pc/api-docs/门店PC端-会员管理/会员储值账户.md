

## 会员储值账户


**接口地址**:`/api/pc/member/account/{memberId}`


**请求方式**:`GET`


**请求数据类型**:`application/x-www-form-urlencoded`


**响应数据类型**:`*/*`


**接口描述**:


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|memberId|会员ID|path|true|integer(int64)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|RMemberAccountVO|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int32)|integer(int32)|
|message||string||
|data||MemberAccountVO|MemberAccountVO|
|&emsp;&emsp;id|账户ID|integer(int64)||
|&emsp;&emsp;memberId|会员ID|integer(int64)||
|&emsp;&emsp;balance|账户余额|number||
|&emsp;&emsp;totalRecharge|累计储值|number||
|success||boolean||


**响应示例**:
```javascript
{
	"code": 0,
	"message": "",
	"data": {
		"id": 0,
		"memberId": 0,
		"balance": 0,
		"totalRecharge": 0
	},
	"success": true
}
```
