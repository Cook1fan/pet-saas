# 验收标准

## 平台管理端

### 登录功能
- [ ] 平台管理员可以使用账号密码成功登录
- [ ] 登录失败有明确错误提示
- [ ] 登录成功后跳转到数据看板

### 租户管理
- [ ] 可以按门店名称、手机号、状态筛选租户列表
- [ ] 可以创建新租户，自动生成账号密码
- [ ] 可以启用/禁用租户账号
- [ ] 可以重置租户管理员密码

### 数据看板
- [ ] 展示总门店数、总会员数、今日订单数、今日 GMV
- [ ] 数据实时更新，无明显延迟

## 门店管理端

### 登录功能
- [ ] 可以选择租户，使用账号密码登录
- [ ] 店员账号无法访问配置类页面
- [ ] 登录成功后跳转到门店首页

### 首页/数据概览
- [ ] 展示今日订单数、今日 GMV
- [ ] 库存预警商品醒目提示
- [ ] 快捷入口可正常跳转

### 会员管理
- [ ] 可以手动录入会员信息
- [ ] 可以按手机号/姓名/标签筛选会员
- [ ] 可以下载 Excel 模板，批量导入会员
- [ ] 可以为会员添加宠物档案
- [ ] 可以设置宠物服务提醒时间

### 储值次卡
- [ ] 可以创建储值规则（储值金额、赠送金额）
- [ ] 可以创建次卡规则（次数、价格、有效期）
- [ ] 可以启用/禁用规则
- [ ] 可以查看核销记录

### 开单收银
- [ ] 可以选择商品/服务
- [ ] 可以选择会员（可选）
- [ ] 可以选择支付方式（微信/现金/储值/次卡）
- [ ] 开单成功后生成订单
- [ ] 库存/储值/次卡实时扣减

### 订单管理
- [ ] 可以按时间/会员/支付方式/状态筛选订单
- [ ] 可以查看订单详情
- [ ] 可以对微信支付/储值订单发起退款
- [ ] 退款后库存/余额自动回退

### 库存管理
- [ ] 可以录入商品，选择分类
- [ ] 可以设置库存预警值
- [ ] 可以手动录入出入库记录
- [ ] 库存低于预警值时在首页显示预警

### 营销活动
- [ ] 可以创建拼团活动，设置价格、成团人数、时间、库存
- [ ] 可以创建秒杀活动
- [ ] 可以生成活动小程序码
- [ ] 可以查看活动参与数据

### AI 助手
- [ ] 可以选择文案类型，输入关键词生成 3 条备选文案
- [ ] 可以复制生成的文案
- [ ] 可以查看剩余 AI 调用次数

### 对账中心
- [ ] 可以查看每笔流水记录
- [ ] 可以按时间筛选对账记录
- [ ] 对账异常有明显标识

## 通用标准

### 性能
- [ ] 页面首次加载时间 ≤ 3 秒
- [ ] 列表分页加载流畅
- [ ] 表单提交响应及时

### 兼容性
- [ ] Chrome 最新版功能正常
- [ ] Edge 最新版功能正常
- [ ] Firefox 最新版功能正常

### 用户体验
- [ ] 所有操作有明确反馈
- [ ] 加载状态有 loading 提示
- [ ] 错误信息清晰易懂
- [ ] 表单有友好的验证提示
