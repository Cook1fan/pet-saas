# 宠物门店私域运营一体化 SaaS - 前端总览

## 项目概述

**pet-saas-admin-pc** 是宠物门店私域运营一体化 SaaS 系统的 PC 管理端前端项目，采用 Vue3 + Element Plus + Vite 技术栈，包含**平台管理端**和**门店管理端**两套管理界面。

## 产品定位

专为社区宠物单门店打造的「多租户隔离 + 全端覆盖 + AI 赋能」轻量化私域运营 SaaS 的 PC 管理前端，提供门店运营、会员管理、营销活动、AI 助手、对账中心等全功能模块。

## 技术栈

| 技术 | 版本 | 用途 |
|-----|------|------|
| Vue3 | 3.4.x | 核心前端框架 |
| Element Plus | 最新稳定版 | UI 组件库 |
| Vite | 最新稳定版 | 构建工具 |
| Pinia | 最新稳定版 | 状态管理 |
| Axios | 最新稳定版 | HTTP 请求库 |
| ECharts | 最新稳定版 | 数据可视化 |
