## 通用约定

### 请求头

| 参数 | 说明 |
|-----|------|
| Content-Type | application/json |
| satoken | 登录 token |

### 响应格式

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 分页请求

```typescript
interface PageQuery {
  page: number      // 页码，从1开始
  pageSize: number  // 每页条数
}
```

### 分页响应

```typescript
interface PageResult<T> {
  list: T[]
  total: number
  page: number
  pageSize: number
}
```
