## 路由设计

### 路由守卫

1. **登录守卫**：未登录用户重定向到登录页
2. **权限守卫**：根据角色控制页面访问权限
3. **租户隔离**：门店端自动携带 tenant_id

### 路由配置示例

```typescript
// router/index.ts
import { createRouter, createWebHistory } from 'vue-router'
import { useUserStore } from '@/stores/user'

const routes = [
  // 平台端登录
  {
    path: '/platform/login',
    name: 'PlatformLogin',
    component: () => import('@/views/platform/login/index.vue'),
    meta: { requiresAuth: false }
  },
  // 平台端布局
  {
    path: '/platform',
    component: () => import('@/views/platform/layout/index.vue'),
    meta: { requiresAuth: true, role: 'platform_admin' },
    children: [
      {
        path: 'dashboard',
        name: 'PlatformDashboard',
        component: () => import('@/views/platform/dashboard/index.vue')
      },
      {
        path: 'tenant',
        name: 'TenantList',
        component: () => import('@/views/platform/tenant/list.vue')
      }
    ]
  },
  // 门店端登录
  {
    path: '/shop/login',
    name: 'ShopLogin',
    component: () => import('@/views/shop/login/index.vue'),
    meta: { requiresAuth: false }
  },
  // 门店端布局
  {
    path: '/shop',
    component: () => import('@/views/shop/layout/index.vue'),
    meta: { requiresAuth: true, role: 'shop' },
    children: [
      {
        path: 'dashboard',
        name: 'ShopDashboard',
        component: () => import('@/views/shop/dashboard/index.vue')
      },
      {
        path: 'member/list',
        name: 'MemberList',
        component: () => import('@/views/shop/member/list.vue')
      }
      // ... 更多路由
    ]
  }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

router.beforeEach((to, from, next) => {
  const userStore = useUserStore()

  if (to.meta.requiresAuth && !userStore.isLoggedIn) {
    next('/shop/login')
  } else {
    next()
  }
})

export default router
```

## 布局组件

### 平台端布局
- 顶部导航：logo、系统名称、用户信息、退出登录
- 左侧菜单：数据看板、租户管理、系统配置

### 门店端布局
- 顶部导航：门店名称、logo、用户信息、退出登录
- 左侧菜单：首页、门店配置、会员管理、储值次卡、开单收银、订单管理、库存管理、营销活动、AI 助手、对账中心
