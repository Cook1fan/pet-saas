# 技术栈与环境

## 核心技术栈

| 技术 | 版本 | 用途 | 选型理由 |
|-----|------|------|---------|
| Vue3 | 3.4.x | 核心前端框架 | Composition API、更好的 TypeScript 支持 |
| TypeScript | 5.x | 类型系统 | 类型安全、提升代码可维护性 |
| Element Plus | 最新稳定版 | UI 组件库 | 开箱即用、适配后台管理系统 |
| Vite | 最新稳定版 | 构建工具 | 极快的热更新、开发体验好 |
| Pinia | 最新稳定版 | 状态管理 | Vue3 官方推荐、API 简洁 |
| Vue Router | 4.x | 路由管理 | 官方路由、支持导航守卫 |
| Axios | 最新稳定版 | HTTP 客户端 | 统一请求封装、拦截器 |
| ECharts | 最新稳定版 | 数据可视化 | 数据看板、趋势图 |

## 开发环境要求

| 工具 | 版本要求 |
|-----|---------|
| Node.js | 18+ |
| npm / pnpm | 最新稳定版 |
| 浏览器 | Chrome / Edge / Firefox 最新 2 个版本 |

## vite.config.ts 配置

```typescript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import { resolve } from 'path'

export default defineConfig({
  plugins: [vue()],
  resolve: {
    alias: {
      '@': resolve(__dirname, 'src')
    }
  },
  server: {
    port: 3000,
    proxy: {
      '/api': {
        target: 'http://localhost:8080',
        changeOrigin: true
      }
    }
  }
})
```

## 环境变量

`.env.development`：
```
VITE_API_BASE_URL=http://localhost:8080/api
VITE_APP_TITLE=宠物门店 SaaS 管理端
```

`.env.production`：
```
VITE_API_BASE_URL=https://api.yourdomain.com/api
VITE_APP_TITLE=宠物门店 SaaS 管理端
```
