# 功能清单

## 功能模块总览

### 平台管理端

| 模块 | 子模块 | 优先级 | 说明 |
|-----|-------|-------|------|
| 登录 | 平台管理员登录 | P0 | 平台超级管理员账号密码登录 |
| 租户管理 | 租户列表 | P0 | 按门店名称、手机号、状态筛选租户 |
| | 租户创建 | P0 | 手动创建租户，生成初始账号密码 |
| | 租户状态管理 | P0 | 启用/禁用租户、重置密码 |
| 数据看板 | 全局数据概览 | P0 | 总门店数、会员数、今日订单数、GMV |
| 系统配置 | AI 额度配置 | P1 | 配置每日免费 AI 调用次数 |
| | 短信模板配置 | P1 | 配置短信发送模板 |

## 优先级说明

- **P0**：MVP 核心功能，必须实现
- **P1**：重要功能，MVP 后期实现
- **P2**：后续迭代功能
