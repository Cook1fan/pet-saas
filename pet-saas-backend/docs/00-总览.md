## 1. 项目概述与核心目标

### 1.1 产品定位

专为社区宠物单门店打造的 **「多租户隔离 + 全端覆盖 + AI 赋能」轻量化私域运营 SaaS**，解决门店「会员流失、不会做活动、私域运营难、对账混乱」的核心痛点，99 元 / 月零门槛使用，帮门店省时间、锁复购、提营收。

### 1.2 目标用户

| 用户类型 | 核心特征 | 核心需求 |
|---------|---------|---------|
| 平台运营方 | 系统开发者 / 管理者 | 多租户隔离管理、轻资产运营、成本可控、可规模化 |
| 门店管理员（老板） | 社区单店经营者，无专业 IT 能力 | 低成本工具、现成营销模板、AI 自动运营、清晰对账 |
| 门店店员 | 门店日常操作人员 | 简单易上手、开单核销快、操作门槛低 |
| C 端宠物主人 | 门店到店 / 线上客户 | 便捷参与活动、查看储值 / 次卡、接收服务提醒、快速咨询 |

### 1.3 核心价值

- **对门店：** 99 元 / 月即可拥有「会员管理 + 营销工具 + AI 运营 + 对账系统」，不用复杂培训，拿来就能用
- **对 C 端用户：** 一站式查看自己的服务权益，便捷参与门店活动，接收贴心的宠物服务提醒
- **对平台方：** 多租户数据 100% 隔离，轻资产运营，不碰门店资金，合规无风险
