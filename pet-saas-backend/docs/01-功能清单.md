## 核心功能清单（按优先级 P0/P1/P2 排序）

### 3.1 平台 PC 管理端（P0）

| 功能模块 | 功能点 | 优先级 | 说明 |
|---------|-------|-------|------|
| 租户管理 | 租户手动创建 | P0 | 平台填写门店名称、老板手机号、地址，系统自动生成唯一 tenant_id、管理员账号/初始密码 |
| 租户管理 | 租户状态管理 | P0 | 启用/禁用租户账号、重置管理员密码 |
| 租户管理 | 租户列表查询 | P0 | 按门店名称、手机号、创建时间、状态筛选租户 |
| 全局数据看板 | 全平台数据概览 | P0 | 展示总门店数、总会员数、今日订单数、今日 GMV、本月 GMV 趋势图 |
| 系统配置 | 全局规则配置 | P1 | 配置 AI 调用额度、短信模板、支付配置 |

### 3.2 门店 PC 管理端（P0）

| 功能模块 | 功能点 | 优先级 | 说明 |
|---------|-------|-------|------|
| 门店基础配置 | 门店信息完善 | P0 | 编辑门店名称、地址、联系方式、营业时间、logo |
| 门店基础配置 | 员工账号管理 | P0 | 创建/编辑/删除店员账号，设置角色（管理员/店员） |
| 门店基础配置 | 支付配置 | P0 | 配置门店自有的微信商户号参数 |
| 会员&宠物管理 | 会员手动录入 | P0 | 录入客户手机号、姓名，自动生成会员档案，支持绑定多只宠物 |
| 会员&宠物管理 | 会员批量导入 | P0 | 提供 Excel 模板下载，支持批量导入会员+宠物信息 |
| 会员&宠物管理 | 会员标签管理 | P1 | 自动生成标签，支持手动添加/编辑标签，按标签筛选会员 |
| 会员&宠物管理 | 宠物档案管理 | P0 | 录入宠物品种、生日、性别、疫苗/驱虫/洗护日期，设置下次提醒时间 |
| 储值&次卡管理 | 规则配置 | P0 | 创建储值规则、次卡规则，支持启用/禁用 |
| 储值&次卡管理 | 核销管理 | P0 | 开单时选择储值/次卡支付，自动扣减余额/次数 |
| 储值&次卡管理 | 记录查询 | P1 | 按时间、会员筛选储值/次卡的购买、核销记录，支持导出 |
| 开单&订单管理 | 开单收银 | P0 | 选择商品/服务、选择会员（可选）、选择支付方式，完成开单 |
| 开单&订单管理 | 订单管理 | P0 | 按时间、会员、支付方式、订单状态筛选订单，查看详情，支持退款 |
| 库存管理 | 商品分类 | P0 | 固定 4 大类 + 细分小类 |
| 库存管理 | 商品录入 | P0 | 录入商品名称、分类、价格、库存、库存预警值 |
| 库存管理 | 出入库管理 | P1 | 手动录入入库/出库记录，选择商品、数量、备注 |
| 库存管理 | 库存预警 | P1 | 库存低于预警值时，后台首页显示红色预警 |
| 营销活动管理 | 拼团活动创建 | P0 | 选择商品/服务、设置拼团价格、成团人数、活动时间、库存、限购数量 |
| 营销活动管理 | 秒杀活动创建 | P0 | 选择商品/服务、设置秒杀价格、活动时间、库存、限购数量 |
| 营销活动管理 | 活动数据查看 | P1 | 展示活动的参与人数、订单数、GMV、拼团成功/失败数 |
| 营销活动管理 | 活动海报生成 | P1 | 选择活动，选择海报风格，调用即梦 API 生成海报 |
| AI 运营助手 | AI 智能提醒 | P1 | 每日定时扫描宠物服务周期，生成个性化提醒话术 |
| AI 运营助手 | AI 文案生成 | P1 | 选择文案类型、输入关键词，一键生成 3 条备选文案 |
| AI 运营助手 | AI 问答配置 | P2 | 配置门店常见问题及标准答案 |
| AI 运营助手 | AI 调用额度管理 | P1 | 每个门店每日免费 50 次 AI 调用，超出后提示额度用完 |
| 对账中心 | 流水记录 | P0 | 每一笔充值、消费、退款生成唯一流水号 |
| 对账中心 | 自动对账 | P1 | 每日凌晨自动调用微信支付账单 API，下载门店前一日账单 |
| 对账中心 | 对账明细 | P1 | 按时间筛选对账记录，查看每一笔流水的系统记录与微信记录 |
| 对账中心 | 异常告警 | P1 | 出现对账异常时，给门店管理员和平台发送短信/系统通知 |

### 3.3 商家版小程序（P0）

| 功能模块 | 功能点 | 优先级 | 说明 |
|---------|-------|-------|------|
| 账号登录 | 账号登录 | P0 | 用门店管理员/店员账号密码登录，自动绑定对应租户 |
| 数据概览 | 数据概览 | P0 | 展示今日订单数、今日 GMV、本月 GMV |
| 订单管理 | 订单管理 | P0 | 查看今日订单列表、订单详情 |
| 次卡核销 | 次卡核销 | P0 | 扫描客户的核销码，完成次卡核销，自动扣减次数 |
| 海报生成 | 海报生成 | P1 | 选择已创建的活动，生成活动海报，保存到手机相册或转发到微信 |

### 3.4 C 端微信小程序（P0）

| 功能模块 | 功能点 | 优先级 | 说明 |
|---------|-------|-------|------|
| 登录&门店绑定 | 登录&门店绑定 | P0 | 扫门店专属小程序码/活动链接进入，微信授权登录，自动绑定对应门店 |
| 多门店切换 | 多门店切换 | P1 | 「我的门店」模块展示已绑定的所有门店，点击切换对应门店的系统 |
| 活动中心 | 活动中心 | P0 | 首页展示门店的拼团/秒杀活动，点击进入详情，选择规格支付，邀请好友拼团 |
| 个人中心 | 个人中心 | P0 | 查看储值余额、次卡剩余次数/有效期、历史订单、宠物档案，生成次卡核销码 |
| AI 客服 | AI 客服 | P1 | 点击进入对话页面，输入问题，AI 自动回答，支持查看历史对话 |
| 服务提醒 | 服务提醒 | P1 | 接收门店发送的疫苗/驱虫/洗护提醒微信模板消息 |
