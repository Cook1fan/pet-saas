## 8. 非功能要求

### 8.1 性能需求

| 指标 | 要求 | 说明 |
|-----|------|------|
| 并发量 | QPS 1000+ | 拼团/秒杀峰值零故障 |
| 库存扣减 | 100% 准确 | 无超卖 |
| 页面响应时间 | ≤2 秒 | 普通接口响应 |
| AI 文案生成响应时间 | ≤5 秒 | 文案生成接口 |
| AI 客服响应时间 | ≤3 秒 | 客服问答接口 |
| 系统可用性 | ≥99.9% | 全年 downtime ≤8.76 小时 |

### 8.2 安全需求

| 需求 | 说明 |
|-----|------|
| 数据传输加密 | 所有数据传输使用 HTTPS 加密 |
| 敏感数据加密存储 | 会员手机号、支付信息采用 AES-256 加密存储 |
| 多租户数据隔离 | 多租户数据 100% 隔离，无跨租户数据访问风险 |
| 操作日志 | 所有后台操作记录日志，可追溯 |
| 微信支付/授权 | 完全符合微信官方安全规范 |
| 接口签名校验 | 接口请求添加签名校验，防止恶意调用（需补充确认）|

### 8.3 日志要求

| 日志类型 | 记录内容 | 保留时间 |
|---------|---------|---------|
| 操作日志 | 用户操作、参数、结果 | 需补充确认 |
| 异常日志 | 异常堆栈、请求参数 | 需补充确认 |
| 支付日志 | 支付请求、回调、结果 | 需补充确认 |
| AI 调用日志 | 调用类型、Prompt、结果、耗时 | 需补充确认 |
