## 5. 接口规范

### 5.1 通用响应格式

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 5.2 异常码说明

| 异常码 | 说明 |
|-------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未登录或 token 失效 |
| 403 | 无权限访问 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
| 1001 | 账号或密码错误 |
| 1002 | 租户已被禁用 |
| 1003 | 库存不足 |
| 1004 | 储值余额不足 |
| 1005 | 次卡次数不足或已过期 |
| 1006 | AI 调用额度已用完 |
| 1007 | 请勿重复下单 |
| 1008 | 对账异常 |
