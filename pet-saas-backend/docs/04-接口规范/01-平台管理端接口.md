### 5.3 平台 PC 管理端接口

#### 5.3.1 租户管理接口

| 接口名称 | 请求方式 | URL | 说明 |
|---------|---------|-----|------|
| 租户手动创建 | POST | /api/platform/tenant/create | 平台创建租户账号 |

**请求参数：**
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| shopName | String | 是 | 门店名称 |
| adminPhone | String | 是 | 老板手机号 |
| address | String | 是 | 门店地址 |

**响应数据：**
| 字段名 | 类型 | 说明 |
|-------|------|------|
| tenantId | Long | 租户 ID |
| username | String | 管理员账号 |
| password | String | 初始密码 |

---

| 接口名称 | 请求方式 | URL | 说明 |
|---------|---------|-----|------|
| 租户状态管理 | POST | /api/platform/tenant/updateStatus | 启用/禁用租户、重置密码 |

**请求参数：**
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| tenantId | Long | 是 | 租户 ID |
| status | Integer | 否 | 状态：0-禁用，1-启用 |
| resetPassword | Boolean | 否 | 是否重置密码 |

---

| 接口名称 | 请求方式 | URL | 说明 |
|---------|---------|-----|------|
| 租户列表查询 | GET | /api/platform/tenant/list | 租户列表分页查询 |

**请求参数：**
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| shopName | String | 否 | 门店名称（模糊搜索） |
| adminPhone | String | 否 | 老板手机号 |
| status | Integer | 否 | 状态 |
| pageNum | Integer | 是 | 页码 |
| pageSize | Integer | 是 | 每页条数 |

**响应数据：**
| 字段名 | 类型 | 说明 |
|-------|------|------|
| total | Long | 总数 |
| list | Array | 租户列表 |

#### 5.3.2 全局数据看板接口

| 接口名称 | 请求方式 | URL | 说明 |
|---------|---------|-----|------|
| 全平台数据概览 | GET | /api/platform/dashboard/overview | 获取全平台核心数据 |

**响应数据：**
| 字段名 | 类型 | 说明 |
|-------|------|------|
| totalShopCount | Integer | 总门店数 |
| totalMemberCount | Long | 总会员数 |
| todayOrderCount | Integer | 今日订单数 |
| todayGmv | BigDecimal | 今日 GMV |
| monthlyGmvTrend | Array | 本月 GMV 趋势数据 |
