### 5.4 门店 PC 管理端接口

#### 5.4.1 登录接口

| 接口名称 | 请求方式 | URL | 说明 |
|---------|---------|-----|------|
| PC 端登录 | POST | /api/pc/login | 门店管理员/店员登录 |

**请求参数：**
| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| username | String | 是 | 账号 |
| password | String | 是 | 密码 |

**响应数据：**
| 字段名 | 类型 | 说明 |
|-------|------|------|
| token | String | 登录 token |
| admin | Object | 管理员信息 |
