## 7. 权限控制方案

### 7.1 Sa-Token 配置

**application.yml 配置：**
```yaml
sa-token:
  token-name: satoken
  timeout: 2592000 # 30天有效期
  is-concurrent: true
  token-prefix:
    pc: pc_
    merchant: merchant_
    user: user_
```

### 7.2 角色权限定义

| 角色 | 所属端 | 权限说明 |
|-----|-------|---------|
| platform_admin | 平台 PC 管理端 | 平台超级管理员，可访问所有租户数据 |
| shop_admin | 门店 PC 管理端/商家版小程序 | 门店管理员，拥有门店全功能权限 |
| shop_staff | 门店 PC 管理端/商家版小程序 | 门店店员，仅能访问开放功能（无配置类、数据导出、财务对账权限）|
| user | C 端微信小程序 | C 端用户，仅能访问自己的数据 |

### 7.3 多端登录隔离

| 端 | 登录接口 | Token 前缀 | Session 存储 |
|----|---------|-----------|-------------|
| 平台 PC 管理端 | /api/platform/login | platform_ | tenant_id, admin_id, role |
| 门店 PC 管理端 | /api/pc/login | pc_ | tenant_id, admin_id, role |
| 商家版小程序 | /api/merchant/login | merchant_ | tenant_id, admin_id, role |
| C 端微信小程序 | /api/user/wxLogin | user_ | tenant_id, member_id, openid |

### 7.4 权限校验注解

| 注解 | 说明 |
|-----|------|
| @SaCheckRole("platform_admin") | 仅平台超级管理员可访问 |
| @SaCheckRole("shop_admin") | 仅门店管理员可访问 |
| @SaCheckRole({"shop_admin", "shop_staff"}) | 门店管理员或店员可访问 |
