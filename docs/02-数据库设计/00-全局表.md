# 数据库设计 - 全局表

---

## 通用字段约定

所有表都包含以下审计字段：

| 字段名 | 类型 | 约束 | 默认值 | 说明 |
|-------|------|------|--------|------|
| id | bigint | PK, NOT NULL, AUTO_INCREMENT | - | 主键 ID |
| create_time | datetime | NOT NULL | CURRENT_TIMESTAMP | 创建时间 |
| create_user | bigint | NULL | NULL | 创建人 ID |
| update_time | datetime | NOT NULL | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 修改时间 |
| update_user | bigint | NULL | NULL | 修改人 ID |
| is_deleted | tinyint | NOT NULL | 0 | 逻辑删除：0-未删除，1-已删除 |

### 全局表（无 tenant_id）
