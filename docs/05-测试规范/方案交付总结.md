## 📋 4. 使用指南

### 4.1 快速开始

1. **安装依赖**
```bash
cd docs/05-测试规范/e2e-script/pc
npm install
npx playwright install
```

2. **配置环境**
```bash
cp .env.example .env
# 编辑 .env 文件配置环境变量
```

3. **运行测试**
```bash
# 运行环境检查
npm run test -- test-env.spec.ts

# 运行所有测试
npm run test

# 调试模式
npm run test:debug

# 生成报告
npm run test:ci
npm run report
```

详细说明请查看 `docs/05-测试规范/快速开始.md`

## 📊 5. 测试覆盖范围

### 已规划的业务流程

#### 拼团活动全流程
- 商家端：创建活动 → 配置规则 → 发布 → 查看数据
- 小程序端：浏览 → 发起/加入 → 支付 → 等待成团 → 查看订单
- 后端：库存 → 支付 → 团状态 → 定时任务 → 退款

#### 储蓄卡次卡全流程
- 商家端：创建规则 → 设置有效期 → 管理库存
- 小程序端：购买 → 查看余额 → 核销 → 查看消费记录

### 异常场景
- 库存不足
- 拼团失败
- 网络异常
- 支付失败
