## 1. 业务概述

### 1.1 核心概念

| 概念 | 说明 |
|------|------|
| 拼团活动 | 门店创建的促销活动，指定商品、拼团价格、成团人数、活动时间、库存、限购 |
| 拼团组 | 用户发起拼团后创建的临时组，包含团长和团员，达到成团人数后拼团成功 |
| 团长 | 第一个发起拼团的用户 |
| 团员 | 加入已有拼团的用户 |
| 成团人数 | 活动配置的达到多少人算拼团成功（比如2人团、3人团、5人团） |
| 拼团有效期 | 发起拼团后多长时间内未成团则自动失败（比如24小时） |

### 1.2 拼团状态流转

```
                    ┌─────────────┐
                    │   待支付    │←── 用户下单未支付
                    └──────┬──────┘
                           │ 支付成功
                    ┌──────▼──────┐
                    │  拼团中     │←── 等待其他团员加入
                    └──────┬──────┘
                           │
              ┌────────────┼────────────┐
              │            │            │
   人数满成团 │            │ 有效期到   │
              │            │            │
    ┌─────────▼──────┐ ┌──▼──────────┐
    │   拼团成功     │ │  拼团失败   │
    └────────────────┘ └─────────────┘
```

### 1.3 业务规则

| 规则 | 说明 |
|------|------|
| 限购规则 | 同一用户在同一活动中最多购买 `limit_num` 件商品 |
| 拼团有效期 | 团长发起拼团后 `24小时` 内未成团，则拼团失败 |
| 库存占用 | 下单时 Redis 预扣库存，支付成功后实际扣减数据库库存 |
| 退款规则 | 拼团中：可申请退款，释放库存；拼团成功：按普通订单退款流程；拼团失败：自动退款 |
| 消息通知 | 拼团成功/失败时，给团内所有用户发送微信模板消息 |
| 重复参团 | 同一用户不能同时加入同一活动的多个拼团组 |
| 团满即止 | 拼团组达到成团人数后，自动标记成功，停止接受新成员 |

## 2. 核心流程设计

### 2.1 创建拼团活动（商家端）

```
商家操作流程：
1. 选择商品（从已上架商品中选择）
2. 填写活动信息：
   - 活动标题
   - 拼团价格
   - 成团人数（2-10人）
   - 活动时间范围
   - 活动库存
   - 限购数量
3. 保存活动，系统自动：
   - 校验商品库存
   - 同步活动库存到 Redis
   - 生成活动小程序码
4. 活动开始后，C端可见

时序图：
商家 → 活动管理页 → 创建活动 → 后端校验 → 保存数据库 → 预热Redis → 返回成功
```

### 2.2 发起拼团（C端用户 - 团长）

```
用户操作流程：
1. 浏览活动列表，选择活动
2. 进入活动详情页，查看：
   - 活动价格/原价
   - 成团人数
   - 剩余库存
   - 正在进行的拼团（可直接加入）
3. 点击「发起拼团」
4. 确认订单，支付
5. 支付成功后：
   - 创建拼团组
   - 创建活动订单
   - 展示「邀请好友参团」页面

核心逻辑：
┌─────────────────────────────────────────────────────────────┐
│ 1. 活动基础校验（存在/时间/状态/库存）                        │
│ 2. 校验用户是否已在该活动的进行中的拼团里                      │
│ 3. Redis 预扣库存（防止超卖）                                 │
│ 4. 获取分布式锁（防止同一用户重复下单）                       │
│ 5. 创建订单（order_info + order_item）                        │
│ 6. 创建活动订单（activity_order，status=0待支付）              │
│ 7. 调用微信支付，返回支付参数                                 │
└─────────────────────────────────────────────────────────────┘
```

### 2.3 加入拼团（C端用户 - 团员）

```
用户操作流程：
1. 通过好友分享的链接/小程序码进入活动页
2. 查看该拼团组的当前进度（已有X人，还差Y人成团）
3. 点击「立即参团」
4. 确认订单，支付
5. 支付成功后：
   - 加入该拼团组
   - 检查是否成团
   - 若成团：更新状态，通知所有人，扣减数据库库存
   - 若未成团：等待其他团员

核心逻辑：
┌─────────────────────────────────────────────────────────────┐
│ 1. 活动基础校验                                                │
│ 2. 拼团组校验（存在/未满员/未结束）                            │
│ 3. 校验用户是否已在该团里                                      │
│ 4. 校验用户是否已在该活动的其他团里                            │
│ 5. Redis 预扣库存                                             │
│ 6. 获取分布式锁                                               │
│ 7. 创建订单                                                   │
│ 8. 创建活动订单（绑定 group_id）                              │
│ 9. 返回支付参数                                               │
└─────────────────────────────────────────────────────────────┘
```

### 2.4 微信支付回调处理

```
支付回调流程：
1. 接收微信回调，验证签名
2. 查询订单，确认待支付
3. 更新订单状态为「已支付」
4. 更新 activity_order 状态为「已支付」
5. 判断是发起拼团还是加入拼团：
   - 发起拼团：创建 activity_group，团长加入，状态为「拼团中」
   - 加入拼团：加入已有 activity_group
6. 检查是否成团：
   - 已达到成团人数：
     - 更新 activity_group 状态为「已成团」
     - 更新该团所有 activity_order 状态为「拼团成功」
     - 扣减数据库库存（从 Redis 预扣同步到数据库）
     - 发送微信模板消息通知所有人
   - 未达到成团人数：
     - 检查团人数，更新团进度
7. 生成流水记录
8. 释放分布式锁
9. 返回微信成功响应
```

### 2.5 拼团失败自动处理（定时任务）

```
定时任务：每分钟扫描过期未成团的拼团组
1. 查询所有满足条件的拼团组：
   - status = 1（拼团中）
   - create_time + 24小时 &lt;= now()
2. 遍历每个拼团组：
   - 更新 activity_group 状态为「拼团失败」
   - 更新该团所有 activity_order 状态为「拼团失败」
   - 回滚 Redis 库存
   - 给团内用户退款：
     - 微信支付：调用微信退款 API
     - 储值支付：退回余额
   - 生成退款流水记录
   - 发送微信模板消息通知用户
```

## 3. 数据库设计补充

### 3.1 新增表：activity_group（拼团组表）

```sql
CREATE TABLE `activity_group` (
  `id` bigint NOT NULL AUTO_INCREMENT COMMENT '主键 ID',
  `tenant_id` bigint NOT NULL COMMENT '租户 ID',
  `activity_id` bigint NOT NULL COMMENT '活动 ID',
  `group_no` varchar(32) NOT NULL COMMENT '拼团组号（唯一）',
  `leader_member_id` bigint NOT NULL COMMENT '团长会员 ID',
  `current_num` int NOT NULL DEFAULT 1 COMMENT '当前人数',
  `target_num` int NOT NULL COMMENT '目标成团人数',
  `status` tinyint NOT NULL DEFAULT 1 COMMENT '状态：1-拼团中，2-拼团成功，3-拼团失败',
  `expire_time` datetime NOT NULL COMMENT '过期时间（发起后24小时）',
  `success_time` datetime DEFAULT NULL COMMENT '成团时间',
  `create_time` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `create_user` bigint DEFAULT NULL COMMENT '创建人 ID',
  `update_time` datetime NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '修改时间',
  `update_user` bigint DEFAULT NULL COMMENT '修改人 ID',
  `is_deleted` tinyint NOT NULL DEFAULT 0 COMMENT '逻辑删除：0-未删除，1-已删除',
  PRIMARY KEY (`id`),
  UNIQUE KEY `uk_group_no` (`group_no`),
  KEY `idx_tenant_activity` (`tenant_id`, `activity_id`),
  KEY `idx_status_expire` (`status`, `expire_time`),
  KEY `idx_leader_member` (`leader_member_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='拼团组表';
```

### 3.2 activity_info 表补充字段

```sql
-- 已有字段，确认存在：
-- group_count int COMMENT '成团人数'

-- 补充字段（如果不存在）：
ALTER TABLE `activity_info`
ADD COLUMN IF NOT EXISTS `group_valid_hours` int NOT NULL DEFAULT 24 COMMENT '拼团有效期（小时）' AFTER `group_count`;
```

### 3.3 activity_order 表调整

```sql
-- 已有字段：
-- group_id bigint COMMENT '拼团 ID（拼团用）'

-- 确保索引存在：
ALTER TABLE `activity_order`
ADD INDEX IF NOT EXISTS `idx_group_id` (`group_id`);
```

### 3.4 ER图

```
activity_info (活动表)
      ↓ 1:N
activity_group (拼团组表)
      ↓ 1:N
activity_order (活动订单表)
      ↓ 1:1
order_info (订单表)
      ↓ 1:N
order_item (订单明细表)
```

    private Long createUser;

    @TableField(fill = FieldFill.INSERT_UPDATE)
    private LocalDateTime updateTime;

    @TableLogic
    private Integer isDeleted;
}

    @Override
    public CreateGroupOrderResp launchGroup(LaunchGroupReq req, Long tenantId, Long memberId) {
        Long activityId = req.getActivityId();
        Integer num = req.getNum();

        // 1. 校验活动
        ActivityInfo activity = activityInfoMapper.selectById(activityId);
        validateActivity(activity, tenantId, num, memberId);

        // 2. 校验用户是否已在该活动的进行中的拼团里
        checkUserOngoingGroup(memberId, activityId);

        // 3. Redis 预扣库存
        String stockKey = RedisKey.ACTIVITY_STOCK.getKey(activityId);
        Long remainStock = stringRedisTemplate.opsForValue().decrement(stockKey, num);
        if (remainStock == null || remainStock &lt; 0) {
            stringRedisTemplate.opsForValue().increment(stockKey, num);
            throw new BusinessException("活动库存不足");
        }

        // 4. 获取分布式锁
        String lockKey = RedisKey.ORDER_USER_ACTIVITY.getKey(memberId, activityId);
        RLock lock = redissonClient.getLock(lockKey);
        try {
            if (!lock.tryLock(3, 10, TimeUnit.SECONDS)) {
                stringRedisTemplate.opsForValue().increment(stockKey, num);
                throw new BusinessException("请勿重复下单");
            }

            // 5. 创建订单
            OrderInfo order = createOrder(activity, num, tenantId, memberId);

            // 6. 创建活动订单（待支付）
            ActivityOrder activityOrder = new ActivityOrder();
            activityOrder.setTenantId(tenantId);
            activityOrder.setActivityId(activityId);
            activityOrder.setOrderId(order.getId());
            activityOrder.setMemberId(memberId);
            activityOrder.setStatus(0); // 待支付
            activityOrderMapper.insert(activityOrder);

            // 7. 调用微信支付
            Map&lt;String, String&gt; payParams = wechatPayService.createMiniProgramPayOrder(order);

            // 8. 返回结果
            CreateGroupOrderResp resp = new CreateGroupOrderResp();
            resp.setOrderId(order.getId());
            resp.setOrderNo(order.getOrderNo());
            resp.setPayAmount(order.getPayAmount());
            resp.setWechatPayParams(payParams);
            return resp;

        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
            stringRedisTemplate.opsForValue().increment(stockKey, num);
            throw new BusinessException("系统繁忙，请稍后重试");
        } finally {
            if (lock.isHeldByCurrentThread()) {
                lock.unlock();
            }
        }
    }

    // ==================== 支付成功回调处理 ====================

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void handlePaySuccess(Long orderId) {
        // 1. 查询订单
        OrderInfo order = orderInfoMapper.selectById(orderId);
        if (order == null || order.getPayStatus() != 0) {
            log.warn("订单不存在或已支付，orderId={}", orderId);
            return;
        }

        // 2. 查询活动订单
        ActivityOrder activityOrder = activityOrderMapper.selectOne(
            new LambdaQueryWrapper&lt;ActivityOrder&gt;()
                .eq(ActivityOrder::getOrderId, orderId)
        );
        if (activityOrder == null) {
            log.warn("活动订单不存在，orderId={}", orderId);
            return;
        }

        // 3. 更新订单状态
        order.setPayStatus(1);
        order.setPayTime(LocalDateTime.now());
        orderInfoMapper.updateById(order);

        // 4. 更新活动订单状态
        activityOrder.setStatus(1); // 已支付
        activityOrderMapper.updateById(activityOrder);

        // 5. 生成流水记录
        flowRecordService.createFlowRecord(order, 2); // 2-消费

        // 6. 判断是发起拼团还是加入拼团
        Long groupId = activityOrder.getGroupId();
        ActivityGroup group;
        if (groupId == null) {
            // 发起拼团：创建拼团组
            group = createActivityGroup(activityOrder, order);
        } else {
            // 加入拼团：加入已有拼团组
            group = joinActivityGroup(activityOrder, groupId);
        }

        // 7. 检查是否成团
        if (group.getCurrentNum() &gt;= group.getTargetNum()) {
            handleGroupSuccess(group, activityOrder.getActivityId());
        }

        // 8. 释放分布式锁
        String lockKey = RedisKey.ORDER_USER_ACTIVITY.getKey(
            activityOrder.getMemberId(),
            activityOrder.getActivityId()
        );
        RLock lock = redissonClient.getLock(lockKey);
        if (lock.isHeldByCurrentThread()) {
            lock.unlock();
        }
    }

    // ==================== 拼团成功处理 ====================

    @Transactional(rollbackFor = Exception.class)
    public void handleGroupSuccess(ActivityGroup group, Long activityId) {
        // 1. 更新拼团组状态
        group.setStatus(2);
        group.setSuccessTime(LocalDateTime.now());
        activityGroupMapper.updateById(group);

        // 2. 更新该团所有活动订单状态
        List&lt;ActivityOrder&gt; orders = activityOrderMapper.selectList(
            new LambdaQueryWrapper&lt;ActivityOrder&gt;()
                .eq(ActivityOrder::getGroupId, group.getId())
        );
        for (ActivityOrder order : orders) {
            order.setStatus(2); // 拼团成功
            activityOrderMapper.updateById(order);
        }

        // 3. 扣减数据库库存
        ActivityInfo activity = activityInfoMapper.selectById(activityId);
        GoodsSku sku = goodsSkuMapper.selectById(activity.getSkuId());
        if (sku.getStock() != -1) { // 非不限库存
            int totalNum = orders.size(); // 每人买1件（简化）
            int updated = goodsSkuMapper.deductStock(sku.getId(), totalNum);
            if (updated == 0) {
                throw new BusinessException("扣减库存失败");
            }
        }

        // 4. 发送微信模板消息通知所有人
        for (ActivityOrder order : orders) {
            Member member = memberMapper.selectById(order.getMemberId());
            if (member != null &amp;&amp; member.getOpenid() != null) {
                wxMpMessageService.sendGroupSuccessMessage(member.getOpenid(), activity, group);
            }
        }
    }

    // ==================== 拼团失败处理（定时任务调用） ====================

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void handleGroupFail(Long groupId) {
        // 1. 查询拼团组
        ActivityGroup group = activityGroupMapper.selectById(groupId);
        if (group == null || group.getStatus() != 1) {
            return;
        }

        // 2. 更新拼团组状态
        group.setStatus(3);
        activityGroupMapper.updateById(group);

        // 3. 查询该团所有活动订单
        List&lt;ActivityOrder&gt; activityOrders = activityOrderMapper.selectList(
            new LambdaQueryWrapper&lt;ActivityOrder&gt;()
                .eq(ActivityOrder::getGroupId, groupId)
        );

        // 4. 处理每个订单
        for (ActivityOrder activityOrder : activityOrders) {
            // 更新活动订单状态
            activityOrder.setStatus(3); // 拼团失败
            activityOrderMapper.updateById(activityOrder);

            // 查询订单
            OrderInfo order = orderInfoMapper.selectById(activityOrder.getOrderId());
            if (order == null) {
                continue;
            }

            // 回滚 Redis 库存
            String stockKey = RedisKey.ACTIVITY_STOCK.getKey(activityOrder.getActivityId());
            stringRedisTemplate.opsForValue().increment(stockKey, 1); // 简化：每人买1件

            // 退款
            if (order.getPayStatus() == 1) {
                refundOrder(order, activityOrder.getActivityId());
            }

            // 发送微信模板消息通知
            Member member = memberMapper.selectById(activityOrder.getMemberId());
            if (member != null &amp;&amp; member.getOpenid() != null) {
                ActivityInfo activity = activityInfoMapper.selectById(activityOrder.getActivityId());
                wxMpMessageService.sendGroupFailMessage(member.getOpenid(), activity, group);
            }
        }
    }

    // ==================== 辅助方法 ====================

    private void validateActivity(ActivityInfo activity, Long tenantId, Integer num, Long memberId) {
        if (activity == null) {
            throw new BusinessException("活动不存在");
        }
        if (!activity.getTenantId().equals(tenantId)) {
            throw new BusinessException("活动不存在");
        }
        if (activity.getType() != 1) { // 1-拼团
            throw new BusinessException("不是拼团活动");
        }
        if (activity.getStatus() != 1) { // 1-进行中
            throw new BusinessException("活动未开始或已结束");
        }
        LocalDateTime now = LocalDateTime.now();
        if (now.isBefore(activity.getStartTime()) || now.isAfter(activity.getEndTime())) {
            throw new BusinessException("活动不在有效时间内");
        }
        // 校验限购
        Integer purchased = getUserPurchasedNum(memberId, activity.getId());
        if (purchased + num &gt; activity.getLimitNum()) {
            throw new BusinessException("超过限购数量");
        }
    }

    private void validateGroup(ActivityGroup group, Long tenantId, Long activityId) {
        if (group == null) {
            throw new BusinessException("拼团不存在");
        }
        if (!group.getTenantId().equals(tenantId)) {
            throw new BusinessException("拼团不存在");
        }
        if (!group.getActivityId().equals(activityId)) {
            throw new BusinessException("拼团不存在");
        }
        if (group.getStatus() != 1) { // 1-拼团中
            throw new BusinessException("拼团已结束");
        }
        if (group.getCurrentNum() &gt;= group.getTargetNum()) {
            throw new BusinessException("拼团已满员");
        }
        if (LocalDateTime.now().isAfter(group.getExpireTime())) {
            throw new BusinessException("拼团已过期");
        }
    }

    private void checkUserInGroup(Long memberId, Long groupId) {
        Long count = activityOrderMapper.selectCount(
            new LambdaQueryWrapper&lt;ActivityOrder&gt;()
                .eq(ActivityOrder::getGroupId, groupId)
                .eq(ActivityOrder::getMemberId, memberId)
                .in(ActivityOrder::getStatus, Arrays.asList(1, 2)) // 已支付/拼团成功
        );
        if (count &gt; 0) {
            throw new BusinessException("您已在该拼团中");
        }
    }

    private void checkUserOngoingGroup(Long memberId, Long activityId) {
        // 查询该用户是否在该活动的进行中的拼团里
        List&lt;ActivityOrder&gt; orders = activityOrderMapper.selectList(
            new LambdaQueryWrapper&lt;ActivityOrder&gt;()
                .eq(ActivityOrder::getActivityId, activityId)
                .eq(ActivityOrder::getMemberId, memberId)
                .in(ActivityOrder::getStatus, Arrays.asList(1, 2)) // 已支付/拼团成功
        );
        if (orders.isEmpty()) {
            return;
        }
        for (ActivityOrder order : orders) {
            if (order.getGroupId() == null) {
                continue;
            }
            ActivityGroup group = activityGroupMapper.selectById(order.getGroupId());
            if (group != null &amp;&amp; group.getStatus() == 1) {
                throw new BusinessException("您已在该活动的拼团中，不能重复参与");
            }
        }
    }

    private Integer getUserPurchasedNum(Long memberId, Long activityId) {
        List&lt;ActivityOrder&gt; orders = activityOrderMapper.selectList(
            new LambdaQueryWrapper&lt;ActivityOrder&gt;()
                .eq(ActivityOrder::getActivityId, activityId)
                .eq(ActivityOrder::getMemberId, memberId)
                .eq(ActivityOrder::getStatus, 2) // 拼团成功
        );
        // 简化：每个订单买1件
        return orders.size();
    }

    private OrderInfo createOrder(ActivityInfo activity, Integer num, Long tenantId, Long memberId) {
        // 查询商品SKU
        GoodsSku sku = goodsSkuMapper.selectById(activity.getSkuId());
        if (sku == null) {
            throw new BusinessException("商品不存在");
        }

        // 创建订单
        String orderNo = OrderNoUtil.generateOrderNo();
        OrderInfo order = new OrderInfo();
        order.setTenantId(tenantId);
        order.setOrderNo(orderNo);
        order.setMemberId(memberId);
        order.setTotalAmount(activity.getOriginPrice().multiply(new java.math.BigDecimal(num)));
        order.setPayAmount(activity.getPrice().multiply(new java.math.BigDecimal(num)));
        order.setPayType(1); // 微信支付
        order.setPayStatus(0); // 待支付
        orderInfoMapper.insert(order);

        // 创建订单明细
        OrderItem item = new OrderItem();
        item.setTenantId(tenantId);
        item.setOrderId(order.getId());
        item.setGoodsId(activity.getGoodsId());
        item.setGoodsName(activity.getTitle());
        item.setNum(num);
        item.setPrice(activity.getPrice());
        orderItemMapper.insert(item);

        return order;
    }

    private ActivityGroup createActivityGroup(ActivityOrder activityOrder, OrderInfo order) {
        ActivityInfo activity = activityInfoMapper.selectById(activityOrder.getActivityId());

        // 创建拼团组
        String groupNo = GroupNoUtil.generateGroupNo();
        ActivityGroup group = new ActivityGroup();
        group.setTenantId(activityOrder.getTenantId());
        group.setActivityId(activityOrder.getActivityId());
        group.setGroupNo(groupNo);
        group.setLeaderMemberId(activityOrder.getMemberId());
        group.setCurrentNum(1);
        group.setTargetNum(activity.getGroupCount());
        group.setStatus(1); // 拼团中
        group.setExpireTime(LocalDateTime.now().plus(activity.getGroupValidHours(), ChronoUnit.HOURS));
        activityGroupMapper.insert(group);

        // 更新活动订单，绑定 group_id
        activityOrder.setGroupId(group.getId());
        activityOrderMapper.updateById(activityOrder);

        return group;
    }

    private ActivityGroup joinActivityGroup(ActivityOrder activityOrder, Long groupId) {
        ActivityGroup group = activityGroupMapper.selectById(groupId);

        // 增加团人数
        group.setCurrentNum(group.getCurrentNum() + 1);
        activityGroupMapper.updateById(group);

        return group;
    }

    private void refundOrder(OrderInfo order, Long activityId) {
        // 更新订单状态为已退款
        order.setPayStatus(2);
        orderInfoMapper.updateById(order);

        // 根据支付方式退款
        if (order.getPayType() == 1) {
            // 微信支付：调用微信退款 API
            wechatPayService.refund(order);
        } else if (order.getPayType() == 3) {
            // 储值支付：退回余额
            memberAccountService.refundBalance(order.getMemberId(), order.getPayAmount());
        }

        // 生成退款流水记录
        flowRecordService.createFlowRecord(order, 3); // 3-退款
    }
}

package com.pet.saas.common.util;

import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;
import java.util.Random;

public class GroupNoUtil {

    private static final DateTimeFormatter FORMATTER = DateTimeFormatter.ofPattern("yyyyMMddHHmmss");
    private static final Random RANDOM = new Random();

    public static String generateGroupNo() {
        String timestamp = LocalDateTime.now().format(FORMATTER);
        int random = RANDOM.nextInt(900000) + 100000; // 6位随机数
        return "GRP" + timestamp + random;
    }
}

package com.pet.saas.job;

import com.pet.saas.entity.ActivityGroup;
import com.pet.saas.mapper.ActivityGroupMapper;

import com.baomidou.mybatisplus.core.conditions.query.LambdaQueryWrapper;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.scheduling.annotation.Scheduled;
import org.springframework.stereotype.Component;

import java.time.LocalDateTime;
import java.util.List;

@Slf4j
@Component
@RequiredArgsConstructor
public class GroupExpireJob {

    private final ActivityGroupMapper activityGroupMapper;
    private final GroupActivityService groupActivityService;

    /**
     * 每分钟执行一次
     */
    @Scheduled(cron = "0 * * * * ?")
    public void processExpiredGroups() {
        log.info("开始执行拼团过期处理任务");

        try {
            // 查询所有过期未成团的拼团组
            List&lt;ActivityGroup&gt; groups = activityGroupMapper.selectList(
                new LambdaQueryWrapper&lt;ActivityGroup&gt;()
                    .eq(ActivityGroup::getStatus, 1) // 拼团中
                    .le(ActivityGroup::getExpireTime, LocalDateTime.now())
            );

            log.info("发现{}个过期拼团组", groups.size());

            for (ActivityGroup group : groups) {
                try {
                    groupActivityService.handleGroupFail(group.getId());
                    log.info("处理拼团失败完成，groupId={}", group.getId());
                } catch (Exception e) {
                    log.error("处理拼团失败异常，groupId={}", group.getId(), e);
                }
            }

        } catch (Exception e) {
            log.error("拼团过期处理任务异常", e);
        }

        log.info("拼团过期处理任务执行完成");
    }
}

### 7.2 活动状态同步任务

```java
package com.pet.saas.job;

import com.pet.saas.entity.ActivityInfo;
import com.pet.saas.mapper.ActivityInfoMapper;
import com.baomidou.mybatisplus.core.conditions.query.LambdaQueryWrapper;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.scheduling.annotation.Scheduled;
import org.springframework.stereotype.Component;
import org.springframework.transaction.annotation.Transactional;

import java.time.LocalDateTime;
import java.util.List;

@Slf4j
@Component
@RequiredArgsConstructor
public class ActivityStatusSyncJob {

    private final ActivityInfoMapper activityInfoMapper;

    /**
     * 每分钟执行一次
     */
    @Scheduled(cron = "30 * * * * ?")
    public void syncActivityStatus() {
        log.info("开始执行活动状态同步任务");

        try {
            LocalDateTime now = LocalDateTime.now();

            // 1. 检查应该开始但未开始的活动
            List&lt;ActivityInfo&gt; toStart = activityInfoMapper.selectList(
                new LambdaQueryWrapper&lt;ActivityInfo&gt;()
                    .eq(ActivityInfo::getStatus, 0) // 未开始
                    .le(ActivityInfo::getStartTime, now)
            );
            for (ActivityInfo activity : toStart) {
                activity.setStatus(1);
                activityInfoMapper.updateById(activity);
                log.info("活动已开始，activityId={}", activity.getId());
            }

            // 2. 检查应该结束但未结束的活动
            List&lt;ActivityInfo&gt; toEnd = activityInfoMapper.selectList(
                new LambdaQueryWrapper&lt;ActivityInfo&gt;()
                    .eq(ActivityInfo::getStatus, 1) // 进行中
                    .le(ActivityInfo::getEndTime, now)
            );
            for (ActivityInfo activity : toEnd) {
                activity.setStatus(2);
                activityInfoMapper.updateById(activity);
                log.info("活动已结束，activityId={}", activity.getId());
            }

        } catch (Exception e) {
            log.error("活动状态同步任务异常", e);
        }

        log.info("活动状态同步任务执行完成");
    }
}
```
