## 1. 整体用户旅程地图

### 1.1 新用户首次进入旅程

```
用户扫码进入小程序
    ↓
检查登录状态（未登录）
    ↓
展示【欢迎页】
    ├─ 顶部：门店Logo + 门店名称
    ├─ 中间：欢迎文案 + 门店特色介绍
    └─ 底部：「一键微信登录」按钮
    ↓
点击「一键微信登录」
    ↓
调用 wx.getUserProfile 获取用户信息
    ↓
调用 wx.login 获取 code
    ↓
请求后端 API：POST /api/miniapp/auth/login
    ├─ 请求参数：{ code, encryptedData, iv, tenantId? }
    └─ 响应数据：{ token, memberInfo, isNewMember }
    ↓
判断是否需要绑定门店
    ├─ 场景1：扫码带有 tenantId → 自动绑定该门店
    ├─ 场景2：搜索进入 → 展示「选择门店」页
    └─ 场景3：已绑定过门店 → 直接进入首页
    ↓
存储 token 到本地 Storage
    ↓
存储 memberInfo 到 Pinia Store
    ↓
进入【首页】
```

### 1.2 老用户回访旅程

```
用户点击小程序进入
    ↓
检查本地 Storage 是否有 token
    ├─ 有 token → 调用验证接口 /api/miniapp/auth/verify
    │   ├─ token 有效 → 进入首页
    │   └─ token 无效 → 清除本地数据，走新用户流程
    └─ 无 token → 走新用户流程
```

## 2. 注册/登录/绑定流程

### 2.1 登录流程时序图

```
小程序端                    后端API                   微信服务器
  │                          │                          │
  ├─ wx.getUserProfile() ──→│                          │
  │←─ 用户信息（avatar,nickname）                      │
  │                          │                          │
  ├─ wx.login() ───────────→│                          │
  │←─ code 返回              │                          │
  │                          │                          │
  ├─ POST /auth/login ─────→│                          │
  │  {code,encryptedData,iv}│                          │
  │                          ├─ code2Session ────────→│
  │                          │←─ openid,session_key ──┤
  │                          │                          │
  │                          │  解密用户信息           │
  │                          │  查询/创建会员         │
  │←─ {token,memberInfo} ───┤                          │
  │                          │                          │
  ├─ 存储 token             │                          │
  ├─ 存储 memberInfo        │                          │
```

#### 交互逻辑

1. **「一键微信登录」按钮点击**
   ```typescript
   // 先检查是否勾选协议
   if (!agreeProtocol) {
     uni.showToast({ title: '请先阅读并同意协议', icon: 'none' })
     return
   }
   
   // 调用微信授权
   uni.getUserProfile({
     desc: '用于完善会员资料',
     success: (profileRes) => {
       const { userInfo } = profileRes
       
       // 获取登录code
       uni.login({
         provider: 'weixin',
         success: (loginRes) => {
           // 调用后端登录接口
           login({
             code: loginRes.code,
             encryptedData: userInfo.encryptedData,
             iv: userInfo.iv,
             tenantId: tenantId // 从扫码参数获取
           }).then((res) => {
             // 保存token
             uni.setStorageSync('token', res.token)
             // 保存用户信息
             useMemberStore().setMemberInfo(res.memberInfo)
             // 跳转首页
             uni.switchTab({ url: '/pages/home/index' })
           })
         }
       })
     }
   })
   ```

2. **《用户协议》链接点击** → 跳转 `pages/protocol/user-agreement`

3. **《隐私政策》链接点击** → 跳转 `pages/protocol/privacy-policy`

#### API 接口

**获取附近门店列表**
```
GET /api/miniapp/shop/nearby
Query: { latitude, longitude, keyword? }
Response:
{
  code: 200,
  data: [
    {
      tenantId: "t001",
      shopName: "XX宠物生活馆",
      shopLogo: "https://xxx/logo.jpg",
      address: "XX路XX号",
      distance: 500, // 距离(米)
      businessHours: "09:00-21:00"
    }
  ]
}
```

**绑定门店**
```
POST /api/miniapp/shop/bind
Body: { tenantId }
Response: { code: 200, message: "绑定成功" }
```

#### 交互逻辑

1. **页面加载时**
   - 获取用户位置 `wx.getLocation`
   - 调用 `getNearbyShops(latitude, longitude)` 获取附近门店

2. **选择门店卡片点击**
   ```typescript
   const bindShop = (shop) => {
     bindShopApi({ tenantId: shop.tenantId }).then(() => {
       useMemberStore().setCurrentShop(shop)
       uni.switchTab({ url: '/pages/home/index' })
     })
   }
   ```

3. **搜索门店**
   - 输入关键词时调用 `getNearbyShops(latitude, longitude, keyword)`

### 2.4 异常场景处理

| 异常场景 | 处理方式 | 提示文案 |
|---------|---------|---------|
| 用户拒绝微信授权 | 提示用户授权，再次点击登录时重新拉起授权 | "需要授权才能登录哦~" |
| 网络请求失败 | 显示网络错误页，提供重试按钮 | "网络开小差了，请重试" |
| 微信登录code过期 | 刷新页面重新获取code | "登录已过期，请重试" |
| 租户被禁用 | 提示门店暂时无法服务，引导联系客服 | "门店正在维护中，请联系客服" |

### 3.2 首页数据结构

#### Pinia Store - useHomeStore

```typescript
interface HomeState {
  bannerList: BannerItem[]
  quickNavList: QuickNavItem[]
  flashSaleList: ActivityItem[]
  groupBuyList: ActivityItem[]
  allActivityList: ActivityItem[]
  currentTab: 'all' | 'group' | 'flash'
}

interface BannerItem {
  id: number
  imageUrl: string
  linkType: 'activity' | 'page' | 'none'
  linkValue: string // activityId 或 pagePath
}

interface QuickNavItem {
  id: number
  icon: string
  name: string
  path: string
}

interface ActivityItem {
  id: number
  type: 'group' | 'flash'
  title: string
  coverImage: string
  originalPrice: number
  activityPrice: number
  stock: number
  status: 'pending' | 'ongoing' | 'ended'
  groupInfo?: {
    currentCount: number
    targetCount: number
    deadline: string
  }
  flashInfo?: {
    endTime: string
    remainingSeconds: number
  }
}
```

```typescript
    currentShop: null,
    isLoggedIn: false
  }),
  
  actions: {
    setMemberInfo(info: MemberInfo) {
      this.memberInfo = info
      this.isLoggedIn = true
    },
    
    setToken(token: string) {
      this.token = token
      uni.setStorageSync('token', token)
    },
    
    setCurrentShop(shop: ShopInfo) {
      this.currentShop = shop
      uni.setStorageSync('currentShop', shop)
    },
    
    logout() {
      this.token = null
      this.memberInfo = null
      this.isLoggedIn = false
      uni.removeStorageSync('token')
    }
  }
})
```
