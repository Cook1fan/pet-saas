## 🎯 开发工作流（新流程）

```
1. 在根目录打开 Claude 窗口
   ↓
2. 基于 docs/ 下的文档统一规划
   ↓
3. 在 docs/01-API契约/ 定义接口
   ↓
4. 并行开发：
   ├─ 后端 Claude：在 pet-saas-backend/ 实现接口
   └─ 前端 Claude：在 pet-saas-admin-pc/ 用 mock 开发
   ↓
5. 联调对接
```
