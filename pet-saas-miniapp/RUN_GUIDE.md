## 运行步骤

### 1. 打开微信开发者工具

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具

### 2. 导入项目

1. 点击 **"导入项目"**
2. 项目名称：`宠物门店SaaS拼团`
3. 目录选择：`C:\Users\cook1\Desktop\pet-saas\pet-saas-miniapp\dist\build\mp-weixin\`
4. AppID：
   - 如果有小程序 AppID，填写真实的 AppID
   - 如果没有，可以选择 **"测试号"**
5. 点击 **导入**

### 3. 预览效果

导入成功后，你将看到完整的拼团小程序：

3. **个人中心**：我的订单、设置等

## 可能遇到的问题

### 1. 图片加载问题
- 使用 Emoji 图标替代本地图片，确保兼容性
- 不会出现图片缺失导致的报错

### 2. 网络请求
- 优先使用真实API，失败时返回模拟数据
- 确保在无网络环境下也能正常显示

### 3. 权限问题
- 使用微信小程序测试号时，部分功能可能受限
- 建议使用真实 AppID 测试完整功能

## 更新说明

如果需要修改代码：

```bash
# 修改代码后重新编译
npm run build:mp-weixin

# 或者使用开发模式
npm run dev:mp-weixin
```

重新编译后，在微信开发者工具中点击 **"编译"** 即可看到最新效果。
