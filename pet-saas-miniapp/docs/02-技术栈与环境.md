# 技术栈与环境

## 开发环境

| 工具 | 版本 | 说明 |
|-----|------|------|
| Node.js | 18+ | 运行环境 |
| HBuilderX | 最新版 | IDE（推荐）或 VS Code |
| 微信开发者工具 | 最新版 | 小程序预览与调试 |

## 技术选型

| 技术 | 版本 | 说明 |
|-----|------|------|
| UniApp | 3.x | 跨端框架 |
| Vue | 3.x | 前端框架 |
| TypeScript | 5.x | 类型系统 |
| Pinia | 2.x | 状态管理 |
| uView Plus | 3.x | UI 组件库 |

## 项目启动

```bash
# 安装依赖
npm install

# 运行到微信小程序
npm run dev:mp-weixin

# 使用 HBuilderX 打开项目，点击「运行」→「运行到小程序模拟器」→「微信开发者工具」
```
